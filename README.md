## 📖 项目简介

这是一个全面的CTF（Capture The Flag）学习笔记仓库，包含了MISC、WEB、Reverse、PWN、应急响应等各个方向的知识总结、工具使用、解题思路和实战WriteUp。适合CTF初学者入门学习和有经验的选手查阅参考。

## 🎯 主要内容

### 🔍 1. 分类知识
- **MISC（杂项）**: 内存取证、流量分析、密码学、隐写术等
- **WEB（Web安全）**: SQL注入、XSS、文件上传、反序列化等
- **Reverse（逆向工程）**: 汇编语言、逆向分析、脱壳技术等
- **PWN（二进制利用）**: 栈溢出、堆利用、ROP、格式化字符串等
- **应急响应**: 安全事件响应、日志分析、恶意代码分析等

### 🛠️ 2. 工具与资源
- 文件签名表和常用工具安装指南
- 各类在线工具使用方法
- 自动化脚本和Payload收集
- 系统操作技巧和命令速查

### 📝 3. WriteUp集合
- 各大CTF平台题目解题思路
- 详细的解题步骤和技术分析
- 实战经验总结和技巧分享

### 📚 4. 学习计划
- 系统化的PWN学习路线
- CS基础课程学习指南
- 阶段性学习目标和检查点

## 🚀 快速开始

### 初学者建议路径：
1. 先阅读 `1-分类知识/` 中的基础知识
2. 查看 `学习计划/` 制定个人学习路线
3. 练习 `7-WP/扩展题目/` 中的入门题目
4. 使用 `3-工具库/` 和 `4-Payloads与脚本/` 提升效率

### 进阶学习者：
1. 深入 `2-专题记录/` 学习特定技术
2. 参考 `7-WP/` 中的高难度题目解析
3. 贡献自己的解题思路和工具脚本
